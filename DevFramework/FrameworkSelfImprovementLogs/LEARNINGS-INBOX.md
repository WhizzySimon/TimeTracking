# Learnings Inbox

**Purpose:** Holding pen for Medium/Low confidence learnings that need more evidence before promotion to JIT rules.

**Promotion criteria:**
- Medium confidence → Promote if observed in 2+ different contexts
- Low confidence → Promote if observed in 3+ different contexts

---

## Format

```markdown
## [Date] [Title]

**Observation:** [what happened]
**Proposed rule/example:** [what to add]
**Target file:** [which JIT rule file]
**Confidence:** Medium / Low
**Occurrences:** 1
**Contexts seen:** [list of different contexts where this appeared]
```
